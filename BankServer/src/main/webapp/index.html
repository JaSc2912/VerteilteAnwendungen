BankServer REST API

Welcome to the Bank Server REST API. This server provides banking services via REST endpoints.

Available Endpoints:

POST
/api/benutzer/login - User login
GET
/api/benutzer - Get all users
GET
/api/bankkonten - Get all bank accounts
GET
/api/kreditantraege - Get all credit applications

Test the API:

You can test the API endpoints using tools like:

  • Postman
  • curl
  • Browser (for GET requests)

Example Login Request:

POST /api/benutzer/login
Content-Type: application/json

{
    "benutzername": "admin",
    "passwort": "admin123"
}
